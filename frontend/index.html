🎓 Project Information

PDF Research Assistant is a Retrieval-Augmented Generation (RAG) based system developed as a Major Project at Chandigarh University.

  • Technology: HTML5, CSS3, JavaScript, Python FastAPI
  • Features: RAG, Citation Support, Hybrid Mode, Voice Input
  • Accuracy: 92% with Hybrid RAG approach
  • Use Case: Academic research, document Q&A, literature review
